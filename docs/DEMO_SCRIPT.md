# DEMO SCRIPT — Presentación MVP (7-10 min)

> Estado: BORRADOR — Se actualizará conforme avancen las waves.

## Audiencia
- Stakeholders del ministerio/gobernación
- Equipo técnico

## Guion

### 1. Introducción (1 min)
- Qué resuelve la plataforma
- Roles: entidad (ministerio) y municipio

### 2. Login y roles (1 min)
- Login como entidad_admin → dashboard de entidad
- Login como municipio_user → dashboard de municipio

### 3. Crear convocatoria (2 min)
- Entidad crea convocatoria con nombre, fechas, requisitos
- Define plantilla MGA (etapas y campos)
- Asigna municipios a la convocatoria

### 4. Diligenciar MGA (2 min)
- Municipio ve convocatoria asignada
- Navega wizard por etapas
- Usa asistente IA para completar campos
- Autosave funciona

### 5. Monitoreo (1 min)
- Entidad ve tablero con progreso de municipios
- Drill-down en un municipio: campos + score

### 6. Asistente IA + RAG (1 min)
- Muestra respuesta del asistente con citaciones
- Muestra audit log de la interacción

### 7. Cierre (1 min)
- Resumen de capacidades
- Próximos pasos

## Dataset de demo
- Convocatoria: "Mejoramiento de vías terciarias 2026"
- Municipios: "San José del Guaviare", "Puerto Asís"
- Plantilla MGA: 4 etapas básicas
